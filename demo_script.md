# Investor Demo Script (90 seconds)

**Slide 1 – Hero (10s)**  
- Show landing hero copy: “AI travel assistant — Shimla POC”.  
- Mention promise: real-time intelligence + hidden gems + offline mode for mountain Wi-Fi.

**Slide 2 – Form Fill (15s)**  
- Select 2 days, low budget, solo traveler, interests: trekking + photography.  
- Highlight interest chips powering 70% of the plan.  
- Hit “Generate Itinerary” and call out loading skeleton (micro-interaction).

**Slide 3 – Itinerary Reveal (20s)**  
- Scroll through Day 1 & Day 2 cards: each has morning/afternoon/evening with travel mode, entry fee, food pairing, interest score.  
- Point to “Why this plan” summary (cost estimate, time efficiency, safety note, hidden gem).  
- Mention “Save offline (JSON)” CTA for patchy connectivity.

**Slide 4 – Alerts & Safety (10s)**  
- Highlight the Alerts panel: road closure, weather advisory, winter carnival event.  
- Explain how the itinerary already references these in safety notes.

**Slide 5 – Chat Assistant (15s)**  
- Open chat modal. Ask “Is Kufri crowded today?” → show deterministic answer referencing Instagram alert.  
- Follow up: “Alternate for Kufri if roads closed” → assistant suggests Green Valley/hidden gem.

**Slide 6 – Admin Loop (10s)**  
- Scroll to Admin panel. Show scraped blog/insta entries.  
- Click “Mark Hidden Gem” on a blog → mention API response.  
- Regenerate itinerary quickly to show the newly boosted hidden gem.

**Slide 7 – Closing CTA (10s)**  
- Summarize roadmap: add 3 destinations next, plug real scraper + vector DB, monetize via premium itineraries + local partners.  
- End with CTA banner: “Pilot with 3 destinations · user testing · monetization ready.”
